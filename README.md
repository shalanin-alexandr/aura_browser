# AURA Browser

Минималистичный веб-браузер с космической темой, написанный на C++ с использованием Qt6 и QtWebEngine.

## Возможности

- Кастомная главная страница с анимированным космическим фоном
- Множественные вкладки с современным дизайном
- История посещений с группировкой по датам
- Закладки
- Сохранение данных между сессиями (история, закладки, открытые вкладки)
- Анимация запуска (splash screen)
- Минималистичный UI с фиолетовыми акцентами

## Требования

- Qt 6
- QtWebEngine
- CMake 3.16+
- Компилятор с поддержкой C++17

## Сборка

```bash
cmake -B build -DCMAKE_PREFIX_PATH=/opt/homebrew/Cellar/qt/6.11.0
cmake --build build
```

## Запуск

```bash
./build/AuraBrowser
```

## Структура

- `BrowserWindow` — главное окно браузера
- `TabWidget` — управление вкладками
- `HistoryManager` — управление историей
- `BookmarkManager` — управление закладками
- `DataManager` — сохранение/загрузка состояния
- `SplashScreen` — экран загрузки
- `resources/` — HTML-шаблоны (главная страница, история, закладки)
